# Roadmap

## v0.1 (current)
- Parse git history
- Connect to reasoning pipeline
- Generate architecture/decision/profile outputs
- Provide local CLI commands

## v0.2
- Add commit-level confidence scores
- Add language-aware tokenization for commit text
- Add optional date-range filters

## v0.3
- Add contributor clustering and ownership map
- Add timeline trend summaries (weekly/monthly)
- Add optional static HTML report export

## v1.0
- Stabilize heuristics and plugin interface
- Add test fixtures for synthetic repositories
- Document extension contracts for custom analyzers

## Guiding constraints
- Keep project lightweight
- Keep execution entirely local
- Avoid lock-in and cloud-only features
- Maintain fork-friendly code structure
